Explicação das tags

Essas grouping tags poderiam ser trocadas por divs, porém usamos outras tags para ue o nosso site possa ser impulsionado pelo algoitmo do google

header

Tag semântica para o cabeçalho

nav

Tag semântica para elementos de navegação. Ex: links de um menu.

main

Tag para o conteúdo principal do site

section

Tag que separa uma seção

article

Separa um artigo

aside

conteudo ao lado que está relacionado

span

esqueci pra que servre

footer

Tag para o rodapé
